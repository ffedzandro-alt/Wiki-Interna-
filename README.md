# Wiki Interna
A Wiki Interna é uma plataforma onde utilizadores autorizados podem aceder, ler, introduzir e editar documentos ou ficheiros de forma simples e organizada.
## Funcionalidades
* Login de utilizadores 
* Cadastro de utilizadores
* Edição de documentos 
* Gestão de ficheiros 
## Tecnologias
* Python 
* HTML 
* SQL 
* CSS
## Instalação
bash
git clone https://github.com/user/projeto.git
cd projeto
pip install -r requirements.txt
